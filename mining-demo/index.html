HTTP relay (browser → relay-server → pool)

This page is a lab console: you can change the relay URL and workers manually. The production-like, no-UI version is miner.html (config in miner-config.js). The lure page embed-lab.html embeds miner.html in a hidden iframe. After mining starts, jobs sync with the relay periodically; use Fetch job first to see if a job is ready.

How to open

  • From the parent of mining-demo, run: python -m http.server 8080 --bind 127.0.0.1
  • Open: http://127.0.0.1:8080/mining-demo/index.html
