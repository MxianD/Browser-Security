HTTP 中继（浏览器 → relay-server → 矿池）

本页通过 GET /api/job、POST /api/submit 与本机 relay-server.js 通信；矿池账号与 Stratum TCP 仅在 Node 进程内。请勿用 file:// 打开，需本地 HTTP 服务。 开始挖矿后会自动定时向中继拉取最新任务（矿池下发新块/新难度时与 relay 保持一致），一般不必手动反复点「拉取任务状态」；仅在未开始挖矿时可用它先看是否就绪。

如何打开本页

  • 在 mining-demo 的上一级目录执行：python -m http.server 8080 --bind 127.0.0.1
  • 浏览器访问：http://127.0.0.1:8080/mining-demo/index.html
