连接矿池（需本地代理）

浏览器无法直连矿池 TCP。先运行 node stratum-proxy.js（需 npm install），再连 WebSocket 与矿池，完成 subscribe / authorize，收到 mining.notify 后即可在上方开始挖矿。

请先用 http:// 打开本页（如 http://127.0.0.1:8080），再在此框填 ws://… 并点「连接代理」。地址栏输入 ws:// 会报错。

如何打开：

  • 在项目上级目录（含 mining-demo 文件夹的那一层）执行：python -m http.server 8080 --bind 127.0.0.1（加 --bind 127.0.0.1 可避免 Windows 下只监听 IPv6 导致 127.0.0.1 出现 ERR_EMPTY_RESPONSE）。浏览器打开：http://127.0.0.1:8080/mining-demo/index.html。
  • 若未加 --bind 且打不开，可试：http://[::1]:8080/mining-demo/index.html。
  • 另开终端：npm install 后 node stratum-proxy.js。
  • 机器发烫或卡顿时：先点「停止」，把 Worker 减为 1、负载限制调低。
  • 矿池若要求 TLS，本代理仅支持明文 TCP，需换端口或自行扩展代理。
