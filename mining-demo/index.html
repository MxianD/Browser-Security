Stratum mining (browser)

Connect to a pool over Stratum only; jobs use double SHA-256 with share target hash < diff1÷difficulty. Open this page via a local HTTP server (see below); otherwise the Worker cannot load scripts. Browser hashrate is negligible; worker count and batch size are capped to avoid overheating the machine.

Connect to pool (local proxy required)

Browsers cannot open raw pool TCP. Run node stratum-proxy.js (after npm install), connect WebSocket to the proxy and TCP to the pool, complete subscribe / authorize, then after mining.notify you can start mining above.

Open this page with http:// first (e.g. http://127.0.0.1:8080), then enter ws://… here and click "Connect proxy". Typing ws:// in the address bar will fail.

How to open:

  • From the parent folder of mining-demo, run: python -m http.server 8080 --bind 127.0.0.1 (--bind 127.0.0.1 avoids Windows binding only IPv6 and 127.0.0.1 showing ERR_EMPTY_RESPONSE). Then open: http://127.0.0.1:8080/mining-demo/index.html.
  • If you skipped --bind and the page fails, try: http://[::1]:8080/mining-demo/index.html.
  • In another terminal: npm install then node stratum-proxy.js.
  • If the machine gets hot or stutters: click Stop, set workers to 1, and lower throttle.
  • If the pool requires TLS, this proxy only supports plain TCP; use another port or extend the proxy.
